## О программе
Данная программа демонстрирует основные возможности языка Rust при работе со структурами (struct) на примере прямоугольника `Rectangle`, как описано в главах 5.1, 5.2 и 5.3 учебника.

Программа иллюстрирует три ключевых момента:
1.  **Определение и создание экземпляров структуры** (`Rectangle` с полями `width` и `height`).
2.  **Реализация методов** для структуры с помощью блока `impl` (методы `area` и `can_hold`).
3.  **Использование ассоциированных функций** как конструкторов (функция `square` для создания квадрата).

## Примеры из кода
*   **Метод `area`** — принимает неизменяемую ссылку на self (`&self`) и вычисляет площадь прямоугольника.
*   **Метод `can_hold`** — показывает, как методу можно передать ссылку на другой экземпляр структуры для сравнения.
*   **Ассоциированная функция `square`** — вызывается через `Rectangle::square(25)` и создает новый экземпляр структуры, где ширина равна высоте.
